게임 플레이 방법

  • ← → 방향키로 바를 움직이세요
  • 공이 닿는 바의 위치에 따라서 공이 움직이는 각도가 변화합니다
  • 스테이지는 총 세 스테이지이며,
  • 쿠키로 벽돌을 모두 깨면 다음 스테이지로 이동합니다
  • 각 쿠키의 능력은 S 키 다운 시 발동 시킬 수 있습니다
  • 각 쿠키의 능력은 스테이지당 한 번 사용 가능합니다
  • 벽돌을 깨면 일정확률로 일정 시간 동안 아이템 효과가 적용됩니다
